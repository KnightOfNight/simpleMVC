Directories...

app - application specific files
	controllers - class definitions for any application controllers
	lib - application-specific libraries
	models - class definitions for any application models
	views - all application views
		<controller> - controller-specific views
cfg - configuration files
html - static HTML files
lib - MVC library files
public - Internet facing files
	css - all CSS files
	img - all image files
	js - all JavaScript files
tmp - temporary files
	cache - data cache files
	logs - log files
	sessions - session data files





Objects

Criteria: used to setup a database select: which columns get selected, what
sort order to use, where clause, etc.  Must be instanticated from a method in
a Controller object.

Example: $criteria = new Criteria(new SomethingModel($this->db));
