		setVariable (line 95)
	
 
	
	

Set a variable to a particular value.  The variable will be available in  the view.


From inside an action method in a Controller object...  
  1.  $this->view->setVariable("somevar", "some string");
    2. 

  ...and when writing the action's view, you will have access to a variable  called $somevar set to the value "some string".
Variable names must be unique, and can store any type of data.
